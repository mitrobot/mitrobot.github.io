大家都知道使用Scratch制作出来的文件是以sb或sb2为后缀的文件，这种文件无法脱离Scratch平台运行和播放，下面有一种方法可以将Scratch文件转化为swf格式的流媒体，可以用视频播放器直接运行，这样是不是就方便多了呢。

具体方法：

1.点击网页下方“Open Scratch File”按钮，选择你要转换的文件。

2.在这里有以下选项其含义分别是

    Full Screen    全屏显示

    Show Green Flag Bar    显示绿旗标志

    Start in Turbo Mode    以加速模式显示

    Automatically Start    自动播放

    Show  Scripts    显示脚本

3.根据需求选择好相应的选项后，点击最下面的“Convert to SWF”按钮。

4.在弹出的窗口中选择swf保存的位置，这样swf就转换成功了。

5.用QQ影音或其他视频播放器打开刚转换好的swf文件。
